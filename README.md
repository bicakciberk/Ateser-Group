## Used Technologies

- Html
- Css
- JavaScript


  
## Run it on your computer

Clone Project

```bash
  git clone https://github.com/bicakciberk/Ateser-Group
```

Go to the project directory

```bash
  cd ateser-group
```

Gerekli paketleri yükleyin

```bash
  npm install
```

Sunucuyu çalıştırın

```bash
  npm run start
```
